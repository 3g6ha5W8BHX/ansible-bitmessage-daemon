ansible-bitmessage-daemon
===============

Bitmessage daemon node for Ubuntu server.

Requirements
------------

This role requires Ansible 2.3 or higher

Role Variables
--------------
Default role variables are stored in the `defaults\main.yml`.  

Dependencies
------------
None

Example Playbook
----------------

```
- hosts: all
  become: yes
  roles:
    - ansible-bitmessage-daemon
```
